#  GenAI Prompt Engineering – LangChain Assignment

##  Overview

This project demonstrates how to build **dynamic and reusable prompt systems** using LangChain instead of hardcoded prompts.

The goal is to design a **Mini Prompt Engine** that converts user input into structured prompts.

---

##  Objectives

* Replace hardcoded prompts using `PromptTemplate`
* Handle multiple dynamic inputs
* Create reusable prompt variations
* Build chat-based prompt systems
* Implement input validation
* Design a modular prompt generator

---

##  Tech Stack

* Python
* LangChain
* Jupyter Notebook / Google Colab

---

##  Tasks Implemented

###  Task 1: PromptTemplate Conversion

Converted static prompts into reusable templates.

###  Task 2: Multi-Input Prompt System

Handled multiple variables:

* topic
* audience
* tone

---

###  Task 3: Prompt Variations Engine

Created different prompt styles:

* Teaching
* Interview
* Storytelling

---

###  Task 4: ChatPromptTemplate

Implemented role-based prompts:

* Teacher
* Interviewer
* Motivator

---

###  Task 5: Input Validation

Validated:

* audience → beginner / intermediate / expert
* tone → formal / casual / fun

---

###  Task 6: Prompt Generator App

Built a dynamic function:

```
generate_prompt(topic, audience, tone, style)
```

---

###  Task 7: Template Reusability

Tested one template across multiple inputs.

---

##  Pipeline Flow

User Input → Validation → Prompt Template → Generated Prompt → Output

---

##  Key Learnings

* Prompt engineering is about **designing systems**, not writing one-time prompts
* Separation of logic and templates improves scalability
* Reusable templates are essential for real-world GenAI applications
