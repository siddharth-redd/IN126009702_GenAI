# AI Resume Screening System

## Objective

Build an AI system to evaluate resumes based on job description.

## Pipeline

Resume → Extract → Match → Score → Explain

## Technologies

* Python
* LangChain
* HuggingFace
* LangSmith

## Results

System tested on:

* Strong Candidate
* Average Candidate
* Weak Candidate

## Debugging

Errors like NameError and incorrect outputs were identified and resolved.

## Note

Local model used due to API limitations.
